PacMan_Gaming_Agent

Run:
1. Install Python 3.10+.
2. Install dependencies:
   pip install -r requirements.txt
3. Start the game:
   python main.py

Controls:
- 1 BFS
- 2 DFS
- 3 UCS
- 4 A*
- 5 Toggle minimax mode
- 6 Toggle Markov ghost mode
- B Benchmark all search algorithms
- SPACE Pause / resume
- R Reset
- ESC Quit
